# 🛒 ShopKart eCommerce App

A modern Flask-based eCommerce web application with admin dashboard, analytics, and product management.

---

## 🚀 Quick Start (2 Steps)

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the app

```bash
python run.py
```

Now open:
http://127.0.0.1:5000
